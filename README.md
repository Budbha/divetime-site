# DiveTime website

Готовый статический сайт для дайв-клуба DiveTime.

## Файлы

- `index.html` — структура сайта
- `styles.css` — весь дизайн и адаптив
- `script.js` — мобильное меню и демонстрационная обработка формы

## Как открыть

Откройте `index.html` в браузере или запустите через Live Server.

## Что заменить позже

- контакты в блоке `Контакты`
- имена и фото инструкторов
- реальные фотографии в галерее
- backend для формы записи
